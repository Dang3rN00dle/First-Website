How to setup Python v3.9.2 and Pycharm101

Installing the Python Software

The first thing you will want to do is install the official python software from Python.org(NOTE: The website should automatically detect your OS, so if you use either one of those, just click the big
yellow button that says "Download Python 3.9.2", and it should begin to install.), which will automatically give you 3 programs: The python IDLE, "Python 3.9(64-bit)", and the "Python 3.9
manuals(64-bit)". You, if I am not mistaken, should be asked how you would like to install the program(s), so just click the top option to easily install it. You can use this "Python IDLE" perfectly fine, but if you want to get the functionality of a python console and other things, you might want to follow the next step:

Installing Pycharm

Installing Pycharm is super easy, so this shouldn't take long.
First, you will want to go to the Pycharm Download Page, and click the "Community Download" button(it should be the black button). Once the package is done installing, you will want to click on it,
and minimize the browser(if you don't see the setup wizard). Just follow the setup wizard instructions:
Click next -> if you want, you can change the install location, but that is up to you. Just click next again once you are okay with the install location -> Choose what launcher you have(32-bit or 64-bit) and
whether or not you would like to install the "JRE x86 by JetBrains"(again it's up to you) then click next -> Then just click on the "Install" button, and it will start installing.
It will most likely ask you if you want to import custom settings from previous Pycharm installations, but you most likely won't have any, so just check "Do not import settings", and click "OK". After that
just read and accept the licence terms and conditions thing, click "Don't Send" on the next screen-thing, and Pycharm will launch.
